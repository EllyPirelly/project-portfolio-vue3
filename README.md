## Tech Stack

- HTML, SCSS, JavaScript

## How to run this?
- clone the repo
- cd into project
- `yarn install` to prepare the project setup
- `yarn serve` to compile and hot-reaload for development, this will generally fire up the server at http://localhost:8000/
- `yarn build` to compile and minify for production
- `yarn lint` to lint and fix files
- `control c` to properly stop the server
